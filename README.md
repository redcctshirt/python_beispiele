### Kommandos

```
# Python-Shell starten (interaktive Modus)
python
python3

# Python-Programm (.py .pyc) starten
python hallowelt.py
python3 hallowelt.py

# Python-Programm in Bytecode (.pyc) umwandeln
python -m compileall hallowelt.py
python3 -m compileall hallowelt.py
```
